# Tutorial: langfuse-python

`langfuse-python`项目是一个用于**追踪、监控和评估AI应用**的Python SDK。它通过**自动捕获大语言模型（LLM）的调用**、**管理媒体文件上传**以及**支持批量评估**等功能，帮助开发者深入了解其AI系统的运行表现、成本和用户体验。该项目旨在提供一个**全面的可观测性解决方案**，以便于调试、优化和改进AI应用。
